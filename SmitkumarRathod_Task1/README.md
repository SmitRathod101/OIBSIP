# Iris Flower Classification

## Project Overview

This project focuses on classifying Iris flowers into three different species using Machine Learning techniques.

The model predicts whether a flower belongs to:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

based on the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

---

## Dataset Information

The Iris dataset consists of 150 samples and 3 flower species.

### Features

* SepalLengthCm
* SepalWidthCm
* PetalLengthCm
* PetalWidthCm

### Target Variable

* Species
